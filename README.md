## Controls

- **Left Arrow:** Move Tetrimino left.
- **Right Arrow:** Move Tetrimino right.
- **Down Arrow:** Accelerate Tetrimino's descent.
- **Up Arrow:** Rotate Tetrimino clockwise.
- **Spacebar:** Instantly drop Tetrimino to the bottom.
